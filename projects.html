Traveler Unraveler

I love to travel, plain and simple. However, as I recently visited some pretty remote places on my honeymoon I learned that it could be helpful to have a mobile currency-converter on hand. Enter Traveler Unraveler. Build on React Native, using Redux to manage state and harnessing the Fixer.io API for rates, this currency converter will make sure you stay on top of your cash!

Honeymoon Cruising'!

I built this blog to memorialize the experiences my wife and I shared on our ~6 week honeymoon. We visited 8 different countries, various cities, islands - it was an all around blast! This project is built on rails and features an Active Admin backend allowing for robust posting capabilities. Embedded images are hosted on Amazon S3, using the Paperclip gem.

Habits

Habits is your catered daily journaling application. When you query the habits of many highly successful people, one common denominator stands out - journaling and taking the time to reflect daily. Habits allows you monitor your habits day by day, by creating questions you'd like to answer based on this habits. Users can provide boolean, rating and text responses to simple SMS prompts which in turn translates into meaningful data depicting a users performance on their designated 'habit'. Habits is a decoupled application featuring a Ruby/Rails and Javascript backend with web application views, as well as a React Native mobile view. Habits incorporates various APIs such as Twilio, GoogleMaps, and WeatherUnderground. Other features include catered statistics and geotracking.
